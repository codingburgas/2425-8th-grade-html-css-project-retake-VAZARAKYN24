This website is not created for commercial purposes. It is part of a school project developed by me, an 8th grade student at PGKPI in Burgas.
You can download the readMe.md file from here.

The main idea of the project is to create a website for a fictional restaurant named "Kehlibar" that includes all the basic components of a modern website: a homepage, an about page, a menu, a contact form, and a user login option.

For the visual part of the site, I used modern color palettes and a background video to add style and dynamics. The side navigation menu on the left helps users browse through the pages easily, while the top bar provides quick access to the main sections.

On the site, you will find:

  • A short history of the restaurant and the idea behind it
  • A menu with offers and prices
  • A login form for users
  • A contact form to reach me

Working on this project helped me improve my skills in web programming and time management. I learned how to structure HTML and CSS code, how to create responsive and modern design, and how to develop a complete website independently.

Thank you for taking the time to visit my project! I would appreciate any feedback through the “Contact” section.
